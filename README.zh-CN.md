## 核心策略手册

### 1. 智能 / 评分钱包跟单

在领袖进入跟单池前评分：胜率、平均 ROI、市场多样性、活跃度、校准度。动态加入/剔除，并在 Telegram/Discord 提醒。

### 2. 选择性信号跟单

对每笔成交评分，只复制高信念信号。

### 3. 全量镜像跟单

简单基线 — 通常还需定仓上限与灰尘过滤。

### 4–6. 定仓：比例 / 固定 / 自适应

| 模式 | 思路 | 示例 |
|------|------|------|
| **比例** | 复制风险份额 | 领袖买 $1,000 → 你买 10% = $100 |
| **固定** | 每信号相同 USD | 每笔合格信号 $50 |
| **自适应** | 计算后钳制 | 算出 $320 → 上限 $100 → 执行 $100 |

### 7–9. 多钱包、共识与出场

跟随 3–7 名专家；可选共识入场；出场镜像 / 独立 TP/SL / 混合。

### 10–15. 过滤、延迟、时间窗与执行质量

品类过滤、内存池低延迟、早/晚时间窗、仅买入 vs 买卖全镜像、限价偏移/滑点、灰尘过滤。
